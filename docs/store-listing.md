## Detailed description

Archive Lookup lets you quickly check whether a web page has been preserved in a web archive.

**Right-click any link**
Right-click a link and choose "Look up on Archive.today" to open the most recent archived snapshot in a new tab. If a snapshot exists, you land on it immediately. If not, the archive service's search page opens so you can create one.

**Minimal permissions**
The extension only requires the `contextMenus` permission. No browsing history, page content, or personal data is collected or transmitted.

**Open source**
The full source code is available on GitHub: https://github.com/phejet/lookup-archive-chrome-extension
